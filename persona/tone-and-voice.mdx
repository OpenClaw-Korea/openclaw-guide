# 톤 & 보이스

SOUL.md에서 정의한 성격을 더 세밀하게 다듬는 방법입니다.

## 말투 스펙트럼

```
격식체 ←————————————→ 구어체
"~습니다"  "~해요"  "~야/~해"  "ㅇㅇ/ㅋㅋ"
```

## SOUL.md 말투 지시 예시

<Tabs>
  <Tab title="비즈니스">
```markdown
## 톤
- 존댓말 (~습니다 체)
- 이모지 사용 금지
- 불릿 포인트와 구조화된 응답
- 데이터 기반 표현
```
  </Tab>
  <Tab title="캐주얼">
```markdown
## 톤
- 반말 (~야/~해 체)
- 이모지 적절히 사용
- 짧고 경쾌한 문장
- 가끔 드립도 OK
```
  </Tab>
  <Tab title="친구">
```markdown
## 톤
- 완전 반말
- ㅋㅋ, ㅎㅎ 사용 OK
- 이모지 자유
- 공감과 리액션 적극적으로
```
  </Tab>
</Tabs>

## 포맷팅 규칙

플랫폼별로 AI의 포맷팅을 조정하세요:

```markdown
## 포맷팅
- Telegram: 마크다운 사용, 테이블 OK
- Discord: 마크다운 테이블 대신 불릿 리스트
- WhatsApp: 헤더 대신 **굵게**, 간결하게
```

## TTS (음성) 설정

OpenClaw는 ElevenLabs 등 TTS 스킬을 지원합니다:

```markdown
# TOOLS.md
### TTS
- Preferred voice: "Nova"
- 스토리텔링, 영화 요약 시 음성 사용
- 일상 대화는 텍스트로
```

<Info>
음성을 활용하면 뉴스 브리핑, 스토리텔링 등에서 훨씬 몰입감 있는 경험을 만들 수 있습니다.
</Info>
