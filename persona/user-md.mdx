title: "USER.md — 사용자 프로필"
description: "AI가 알아야 할 당신의 정보"
---

USER.md는 AI가 당신에 대해 알아야 할 기본 정보를 담습니다.
AI가 매 세션마다 이 파일을 읽어서 맞춤형 응답을 제공합니다.

## 기본 구조

```markdown
# USER.md

## 기본 정보
- 이름: 김철수
- 위치: 서울, 대한민국 (KST, UTC+9)
- 직업: 소프트웨어 엔지니어

## 선호
- 간결한 커뮤니케이션 선호
- 불필요한 확인 질문 싫어함
- 아침형 인간 (6시 기상, 23시 취침)
- 커피 좋아함, 차 싫어함

## 기술 스택
- TypeScript, Python
- React, Next.js
- AWS, Docker

## 관심사
- AI/ML 트렌드
- 생산성 도구
- 등산, 독서
```

<Info>
USER.md에 적힌 정보는 AI의 모든 응답에 영향을 미칩니다.
"서울"이라고 적으면 날씨 질문에 자동으로 서울 날씨를 알려줍니다.
</Info>

## 왜 중요한가?

| USER.md 없을 때 | USER.md 있을 때 |
|----------------|----------------|
| "어디 계세요?" 매번 물어봄 | 서울 기준으로 바로 답변 |
| 일반적인 기술 설명 | TypeScript 예시로 설명 |
| "몇 시에 일어나세요?" | 알아서 6시 알람 제안 |

## 보안 주의사항

<Warning>
USER.md는 메인 세션(1:1 대화)에서만 사용됩니다.
그룹 채팅이나 공유 채널에서는 로드되지 않아 개인정보가 보호됩니다.
</Warning>

## 점진적 업데이트

처음부터 완벽할 필요 없습니다. 대화하면서 자연스럽게 추가하세요:

```
나: "나 참고로 글루텐 프리 식단 해"
AI: (USER.md에 식단 선호 추가)
```

<Tip>
"이거 기억해" 라고 말하면 AI가 알아서 USER.md나 MEMORY.md에 기록합니다.
</Tip>
