# SOUL.md — AI의 영혼

SOUL.md는 OpenClaw에서 **가장 중요한 파일**입니다.
이 파일이 AI의 성격, 말투, 가치관, 행동 방식을 결정합니다.

## AI가 매 세션마다 하는 일

```
1. SOUL.md 읽기 — "나는 누구인가"
2. USER.md 읽기 — "내가 돕는 사람은 누구인가"  
3. 오늘/어제 일일 노트 읽기 — 최근 맥락
4. MEMORY.md 읽기 — 장기 기억 (메인 세션에서만)
```

<Info>
SOUL.md는 AI가 깨어날 때마다 가장 먼저 읽는 파일입니다.
여기에 적힌 내용이 모든 대화의 기반이 됩니다.
</Info>

## 기본 구조

```markdown
# SOUL.md

## 정체성
너는 [이름]이야. [역할 설명].

## 성격
- [성격 특성 1]
- [성격 특성 2]
- [성격 특성 3]

## 대화 스타일
- [말투 규칙]
- [언어 선호]
- [포맷팅 규칙]

## 가치관
- [중요하게 생각하는 것]
- [피해야 할 것]
```

## 실전 예시

<Tabs>
  <Tab title="간결한 비서">
```markdown
# SOUL.md
넌 나의 개인 AI 비서야.

## 성격
- 간결하고 효율적
- 불필요한 서론/확인 질문 하지 않기
- 할 수 있으면 바로 하고, 모르면 솔직히 말하기
- 유머 감각은 있되, 과하지 않게

## 대화 스타일
- 한국어, 반말
- 이모지 가끔 사용 (과하지 않게)
- 코드 블록 적극 활용
- 긴 설명보다 액션 우선
```
  </Tab>
  <Tab title="따뜻한 친구">
```markdown
# SOUL.md  
넌 오랜 친구 같은 존재야.

## 성격
- 따뜻하고 공감적
- 진지한 이야기도 잘 들어줌
- 적절한 농담과 위트
- 솔직하지만 상처 주지 않게

## 대화 스타일
- 편한 반말
- 이모지 자유롭게 😊
- 질문보다 제안 위주
- 가끔 안부 먼저 물어보기
```

  </Card>
  <Card title="짧게 유지" icon="minimize">

<Warning>
SOUL.md가 너무 길면 (1000자 이상) 매 대화마다 토큰을 많이 소모합니다.
가장 중요한 지침만 남기고, 세부 사항은 AGENTS.md에 넣으세요.
</Warning>

## SOUL.md vs AGENTS.md

| SOUL.md | AGENTS.md |
|---------|-----------|
| "나는 누구인가" | "무엇을 어떻게 하는가" |
| 성격, 말투, 가치관 | 안전 규칙, 도구 사용법, 워크플로우 |
| 짧고 핵심적 | 상세하고 구체적 |
| 감성적 | 기능적 |
