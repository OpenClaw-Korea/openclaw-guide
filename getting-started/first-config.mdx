설치가 끝났다면 워크스페이스를 설정할 차례입니다.

## 온보딩 마법사

가장 쉬운 방법은 온보딩 마법사를 사용하는 것입니다:

```bash
openclaw onboard
```

<Steps>
  <Step title="AI 모델 선택">
    Anthropic Claude를 권장합니다. API 키를 준비하세요.
  </Step>
  <Step title="채널 선택">
    연결할 메신저를 선택합니다. Telegram이 가장 쉽습니다.
  </Step>
  <Step title="워크스페이스 경로">
    기본값 `~/.openclaw/`을 사용하거나 커스텀 경로를 지정합니다.
  </Step>
  <Step title="SOUL.md 초기화">
    AI의 기본 성격을 설정합니다. 나중에 수정 가능합니다.
  </Step>
</Steps>

## 수동 설정

마법사 대신 수동으로 설정하려면:

### config.yaml

```yaml
# ~/.openclaw/config.yaml
model:
  provider: anthropic
  model: claude-opus-4-6
  thinkingLevel: low

gateway:
  port: 18789
  host: 127.0.0.1

channels:
  telegram:
    token: "YOUR_BOT_TOKEN"
    dm:
      allowFrom:
        - "YOUR_USER_ID"

workspace:
  path: "~/.openclaw"
  
heartbeat:
  enabled: true
  intervalMinutes: 30
```

### SOUL.md 작성

```markdown
# SOUL.md
너는 나의 개인 AI 비서야.
- 한국어로 대화
- 반말 사용
- 간결하고 실용적으로
- 유머 감각 있게
- 불필요한 서론 생략
```

### USER.md 작성

```markdown
# USER.md
- 이름: [당신의 이름]
- 직업: [당신의 직업]
- 위치: 서울, 대한민국 (KST, UTC+9)
- 관심사: [당신의 관심사]
- 선호: 간결한 커뮤니케이션, 불필요한 확인 질문 싫어함
```

## 첫 대화 테스트

<Tabs>
  <Tab title="Telegram">
    1. 봇을 검색하고 `/start` 입력
    2. "안녕, 넌 누구야?" 전송
    3. AI가 SOUL.md에 맞는 성격으로 응답하면 성공!
  </Tab>
  <Tab title="WebChat">
```bash
openclaw webchat
# 브라우저에서 http://127.0.0.1:18789 접속
```
  </Tab>
</Tabs>

<Tip>
첫 대화에서 AI가 너무 딱딱하거나 너무 가벼우면 SOUL.md를 수정하세요.
"좀 더 편하게 말해줘" 같은 피드백도 AI가 즉시 반영합니다.
</Tip>
