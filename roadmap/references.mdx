## 핵심 개념 문서

빠르게 찾아보는 핵심 파일들:

### 설정 파일

```
~/clawd/
├── AGENTS.md      # AI 행동 규칙 (가장 중요!)
├── SOUL.md        # AI 성격 정의
├── USER.md        # 사용자 프로필
├── TOOLS.md       # 로컬 도구 설정
├── MEMORY.md      # 장기 기억
├── HEARTBEAT.md   # 하트비트 체크리스트
└── memory/        # 일일 기록
    └── YYYY-MM-DD.md
```

### CLI 치트시트

```bash
# Gateway
openclaw gateway start|stop|restart|status

# 크론잡
openclaw cron list
openclaw cron add --name "이름" --cron "표현식" --message "프롬프트"
openclaw cron remove --name "이름"

# 스킬
openclaw skill list
openclaw skill install <name>

# 노드
openclaw node status
openclaw node approve <id>
```

## 관련 도구

OpenClaw와 함께 자주 사용하는 외부 도구들:

| 도구 | 설치 | 용도 |
|------|------|------|
| gog | `pip install gog` | Gmail, Calendar, Drive |
| imsg | `brew install nicklama/tap/imsg` | iMessage 제어 |
| remindctl | `brew install nicklama/tap/remindctl` | Apple 미리알림 |

## 문제 해결

막혔을 때 확인할 순서:

1. **공식 문서** — [docs.openclaw.ai](https://docs.openclaw.ai)에서 검색
2. **GitHub Issues** — 같은 문제를 겪은 사람이 있는지 확인
3. **Discord #help** — 커뮤니티에 질문
4. **OpenClaw에게 질문** — "이 에러 뭐야?" 라고 물어보기

<Tip>
OpenClaw 자체에게 OpenClaw 사용법을 물어볼 수 있습니다.
"크론잡 어떻게 설정해?"라고 물어보면 바로 알려줍니다.
</Tip>
