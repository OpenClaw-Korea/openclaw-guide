기본에 익숙해졌다면, 이제 OpenClaw의 진짜 힘을 활용할 차례입니다.
스킬 설치, 자동화, 멀티 디바이스 연동까지 1-3개월 과정입니다.

## 1개월차: 스킬 확장

**목표:** 스킬을 설치하고 도구 생태계를 확장한다.

- [ ] `openclaw skill list` — 사용 가능한 스킬 확인
- [ ] 이메일 연동 (gog CLI): "안 읽은 이메일 확인해줘"
- [ ] 캘린더 연동: "이번 주 일정 보여줘"
- [ ] iMessage 연동 (imsg CLI)
- [ ] Apple Reminders 연동 (remindctl CLI)

### 추천 스킬 설치 순서

```
1. google-tools (이메일 + 캘린더) — 가장 많이 쓰게 됨
2. imsg (iMessage) — macOS 사용자라면 필수
3. remindctl (미리알림) — 할 일 관리
```

<Info>
스킬은 하나씩 추가하고, 충분히 익숙해진 뒤 다음 스킬로 넘어가세요.
한 번에 다 설치하면 혼란만 커집니다.
</Info>

## 2개월차: 자동화

**목표:** 크론잡과 하트비트로 반복 작업을 자동화한다.

### 크론잡 시작하기

```bash
# 아침 브리핑
openclaw cron add --name "morning-briefing" \
  --cron "0 8 * * *" \
  --message "아침 브리핑 해줘" \
  --announce --session isolated

# 크론잡 목록
openclaw cron list

# 크론잡 삭제
openclaw cron remove --name "morning-briefing"
```

### 하트비트 활용

```markdown
# HEARTBEAT.md
- 안 읽은 이메일 체크 (2시간마다)
- 오늘 캘린더 이벤트 리마인더
- iMessage 새 메시지 체크
```

### 추천 자동화

| 자동화 | 주기 | 설명 |
|--------|------|------|
| 아침 브리핑 | 매일 8시 | 일정 + 이메일 + 날씨 |
| 이메일 체크 | 2시간마다 | 중요 이메일 알림 |
| 주간 회고 | 매주 금요일 | 한 주 요약 자동 생성 |

## 3개월차: 노드 연동

**목표:** 여러 디바이스를 연결해서 통합 환경을 만든다.

- [ ] iPhone/iPad 노드 페어링
- [ ] 카메라 스냅 테스트: "현관 카메라 보여줘"
- [ ] 위치 조회: "지금 어디야?"
- [ ] 원격 알림: "폰으로 알림 보내줘"

```bash
# 노드 상태 확인
openclaw node status

# 노드 페어링 승인
openclaw node approve <device-id>
```

<Tip>
노드 연동은 같은 네트워크에서 시작하면 쉽습니다.
먼저 로컬에서 테스트하고, 외부 네트워크로 확장하세요.
</Tip>

## 체크포인트

3개월이 끝나면:

- ✅ 이메일·캘린더·메시지를 자연어로 관리한다
- ✅ 최소 2개의 크론잡이 돌아가고 있다
- ✅ 하트비트가 주기적으로 상태를 체크한다
- ✅ 다른 디바이스에서도 OpenClaw에 접근한다
