# 메모리 검색

시간이 지나면 일일 노트가 쌓입니다. 필요한 정보를 효율적으로 찾는 방법입니다.

## AI에게 직접 물어보기

가장 간단한 방법은 AI에게 질문하는 것입니다:

```
"지난주에 한 프로젝트 A 관련 결정이 뭐였지?"
"2월 초에 설치한 스킬 이름이 뭐였어?"
"최근에 내가 좋다고 했던 레스토랑 어디였지?"
```

AI는 일일 노트와 MEMORY.md를 검색해서 답변합니다.

## CLI로 검색

```bash
# memory 폴더에서 grep 검색
grep -r "프로젝트 A" memory/

# 특정 기간의 노트 검색
grep -r "레스토랑" memory/2026-02-*.md

# 최근 7일 노트 보기
ls -la memory/ | tail -7
```

<Info>
AI도 내부적으로 `grep`, `cat` 등의 명령어로 파일을 검색합니다.
복잡한 검색은 AI에게 맡기는 것이 편합니다.
</Info>

## 메모리 정리 요청 예시

```
"이번 달 일일 노트를 검토하고 MEMORY.md를 업데이트해줘.
중요한 결정, 교훈, 진행 중인 프로젝트만 남기고
나머지는 요약해서 정리해."
```

<Tip>
하트비트에 주기적 메모리 정리를 포함시키면 자동으로 관리됩니다.
```markdown
# HEARTBEAT.md
- [ ] 주 1회: 일일 노트 검토 → MEMORY.md 업데이트
```
</Tip>
