2026년, AI는 더 이상 브라우저에서 열어보는 도구가 아닙니다.
**메신저에서 항상 대기하고, 기억하고, 스스로 판단하여 행동하는 존재** — 그것이 OpenClaw입니다.

이 가이드는 OpenClaw를 처음 접하는 사람부터 고급 자동화를 구축하는 파워유저까지,
**60가지 이상의 실전 팁**을 체계적으로 정리한 결정판입니다.

## "도구"가 아니라 "동거인"

<Info>
  ChatGPT는 택시입니다 — 필요할 때 부릅니다.
  OpenClaw는 룸메이트입니다 — 항상 거기 있습니다.
</Info>

| | ChatGPT / Claude.ai | OpenClaw |
|---|---|---|
| **접근** | 브라우저 열기 | 메신저에서 DM |
| **기억** | 매번 설명 | MEMORY.md로 장기 기억 |
| **행동** | 물어봐야 답함 | 하트비트로 먼저 연락 |
| **성격** | 범용 AI | SOUL.md로 나만의 페르소나 |
| **확장** | 플러그인 | 스킬 + 노드 + 서브에이전트 |
| **실행** | 클라우드 | 내 맥/서버에서 로컬 |

## 빠른 시작

<Steps>
  <Step title="설치 (5분)">
    ```bash
    npm install -g openclaw
    openclaw init
    ```
  </Step>
  <Step title="Telegram 연동 (3분)">
    ```bash
    openclaw config telegram --token YOUR_BOT_TOKEN
    openclaw gateway start
    ```
  </Step>
  <Step title="첫 대화">
    Telegram에서 봇에게 DM: **"안녕, 넌 누구야?"**
  </Step>
  <Step title="페르소나 설정 (선택)">
    `SOUL.md`를 열어 AI의 이름, 성격, 말투를 바꿔보세요.
  </Step>
</Steps>

<Tip>
  설치만 하면 바로 쓸 수 있습니다. 하지만 **SOUL.md와 MEMORY.md를 설정하는 순간**, 
  "범용 챗봇"이 **"나만의 AI"**로 바뀝니다. Part 3부터 읽어보세요.
</Tip>
