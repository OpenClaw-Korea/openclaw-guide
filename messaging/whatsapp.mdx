WhatsApp은 전 세계적으로 가장 많이 쓰이는 메신저입니다.
OpenClaw와 연동하면 WhatsApp에서도 AI와 대화할 수 있습니다.

## 설정

<Steps>
  <Step title="별도 번호 준비">
    **별도의 전화번호를 사용하세요.**
    기본 번호를 사용하면 차단될 위험이 있습니다.
  </Step>
  <Step title="WhatsApp Web 연동">
```bash
openclaw channels login whatsapp
```
QR 코드가 표시됩니다.
  </Step>
  <Step title="QR 코드 스캔">
    WhatsApp → 연결된 기기 → 기기 연결 → QR 코드 스캔
  </Step>
  <Step title="연결 확인">
```bash
openclaw gateway restart
```
  </Step>
</Steps>

<Warning>
WhatsApp은 **자동화 봇을 공식적으로 허용하지 않습니다** (Business API 제외).
비공식 연동이므로 번호 차단 위험이 있습니다.
반드시 별도 번호를 사용하세요.
</Warning>

## 사용법

WhatsApp으로 메시지를 보내면 AI가 응답합니다.

```
사용자: "오늘 날씨 어때?"
AI: "서울 현재 5°C, 맑음이에요. 오후부터 구름 많아져요. 🌤️"
```

### 이미지 전송

사진을 보내면 AI가 분석합니다.

### 음성 메시지

음성을 보내면 텍스트로 변환 후 응답합니다.

## 포맷팅 주의사항

<Warning>
WhatsApp에서는 마크다운 **헤더(#)가 지원되지 않습니다**.
대신 **굵은 글씨** 또는 대문자로 구분하세요.
</Warning>

```markdown
❌ WhatsApp에서 안 되는 것:
# 제목
## 소제목
| 테이블 |

✅ 이렇게 해야 함:
*제목*

• 항목 1
• 항목 2

_이탤릭_, *굵게*, ~취소선~
```

## 그룹 채팅

WhatsApp 그룹에서도 AI를 사용할 수 있습니다.
봇 번호를 그룹에 추가하면 됩니다.

```markdown
# AGENTS.md 규칙 적용
- 멘션되거나 질문받을 때만 응답
- 잡담에는 HEARTBEAT_OK
```

## Telegram과의 비교

| 기능 | Telegram | WhatsApp |
|------|----------|----------|
| 공식 봇 API | ✅ | ❌ (비공식) |
| 차단 위험 | 없음 | 있음 |
| 설정 난이도 | 쉬움 | 보통 |
| 파일 크기 | 50MB | 16MB |
| 포맷팅 | 풍부 | 제한적 |
| 세션 유지 | 영구 | 재연결 필요 |

<Tip>
WhatsApp은 이미 WhatsApp을 주 메신저로 쓰는 경우에 유용합니다.
새로 시작한다면 Telegram을 권장합니다.
</Tip>

## 세션 유지

WhatsApp Web 세션은 주기적으로 끊어질 수 있습니다.
끊어지면 QR 코드를 다시 스캔해야 합니다.

```bash
# 상태 확인
openclaw gateway status

# 재연결 필요 시
openclaw channels login whatsapp
```

<Info>
세션이 자주 끊어진다면 하트비트에서 연결 상태를 모니터링할 수 있습니다.
</Info>
