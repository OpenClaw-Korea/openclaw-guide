# 그룹 채팅 팁

AI를 그룹 채팅에 참여시키면 **유용한 팀 멤버**가 될 수 있지만,
잘못 설정하면 짜증나는 존재가 됩니다. 핵심은 **절제**입니다.

## 황금 규칙: 참여하되, 지배하지 마

```markdown
# AGENTS.md 그룹 채팅 규칙

### 응답할 때:
직접 멘션되거나 질문받을 때
진짜 가치를 더할 수 있을 때
중요한 잘못된 정보 수정
요약 요청

### 조용히 있을 때:
인간들끼리 수다
이미 답변된 질문
"응", "좋아" 수준의 응답만 가능할 때
대화가 잘 흘러갈 때
```

<Info>
인간의 규칙: 인간들도 그룹 채팅에서 모든 메시지에 답하지 않습니다.
AI도 마찬가지여야 합니다. **양보다 질.**
</Info>

## 응답 판단 플로우차트

```
메시지 수신
  ├─ 직접 멘션? → 응답
  ├─ 질문인가? → 내가 잘 아는 분야? → 응답
  ├─ 잘못된 정보? → 중요한 건? → 정정
  ├─ 재밌는 상황? → 자연스러운 한마디? → 응답
  └─ 그 외 → HEARTBEAT_OK (조용히)
```

## 리액션 활용

텍스트 응답 대신 **이모지 리액션**으로 가볍게 참여:

```markdown
### 리액션이 적절한 순간:
감사하지만 답장 불필요 → 👍 ❤️ 🙌
웃긴 메시지 → 😂 💀
흥미로운 정보 → 🤔 💡
승인/확인 → ✅ 👀

### 규칙:
메시지당 리액션 최대 1개
가장 적합한 것 하나만 선택
```

<Tip>
리액션은 "봤어, 인정해"를 채팅 어지럽히지 않고 전달하는 최고의 방법입니다.
텍스트 응답보다 리액션이 적절한 순간이 훨씬 많습니다.
</Tip>

## 플랫폼별 그룹 설정


  ### Telegram
```bash
# 그룹 허용
openclaw config set telegram.groups.allowed "[-100123456789]"
```
BotFather에서 Privacy Mode 해제 필요
그룹 관리자가 봇을 추가
  
  ### Discord
```bash
# 특정 채널만 허용
openclaw config set discord.channels.allowed "[123456789]"
```
봇 역할의 채널 권한 설정
멘션 트리거 권장
  
  ### WhatsApp
봇 번호를 그룹에 초대
자동 응답 빈도 주의 (차단 위험)
  


## 트리플 탭 방지

<Warning>
같은 메시지에 여러 번 다른 형태로 응답하지 마세요.

❌ 나쁜 예:
1. 리액션 😂
2. "ㅋㅋㅋ 웃기다"
3. 관련 밈 공유

✅ 좋은 예:
리액션 😂 (끝)
또는 짧은 응답 하나 (끝)
</Warning>

## SOUL.md에서 그룹 페르소나 설정

```markdown
# SOUL.md

## 그룹 채팅에서의 나
조용한 편. 필요할 때만 말한다.
정보 제공과 유머를 적절히 섞는다.
대화를 지배하려 하지 않는다.
리액션을 적극 활용한다.
```
